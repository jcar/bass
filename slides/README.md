# Strikezone — "Your LLM Is a Compiler" deck

## Run

```
npm install
npm run dev
```

Opens at http://localhost:3030. Presenter view at http://localhost:3030/presenter.

## Export PDF backup

```
npm run export
```

Produces `slides-export.pdf`.
